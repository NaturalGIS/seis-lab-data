# Domain Acronyms

These are some domain-related acronyms, as provided by IPMA

| acronym               | meaning |
|-----------------------| ------- |
| `bathy`	               | campanhas de batimetria |
| `bstk`	               | brutestack, linhas sísmicas com unidades verticais em tempo practicamente sem processamento, produto de QC |
| `ctd`	                | amostrador de conductivity, temperature and depth |
| `dpt`	                | linhas sísmicas com unidades verticais métricas |
| `dpr`	                | daily progress report |
| `dtm`	                | digital terrain model |
| `gis`		               | geographical information system |
| `ks`	                 | projecto Kingdom Suite |
| `mbes`                | multibeam echo sounder, batimetria multifeixe |
| `mig`	                | linhas sísmicas com unidades verticais em tempo com sem migração |
| `mmo`	                | marine mammal observer |
| `mob`	                | mobilization |
| `mul`	                | linhas sísmicas com unidades verticais em tempo com atenuação de multiplo (sem migração) |
| `multichannel seismic` | sismica multicanal |
| `owf`	                | offshore wind farm, bloco de terreno para exploração de energia eólica |
| `poststk`	            | dominio pos-stack para dados sísmicos |
| `prestk`	             | domínio pre-stack para dados sismicos |
| `qas`	                | dados processados de magnetometro |
| `qc`	                 | quality control |
| `res`	                | dados processados de magnetometro |
| `rotv`	               | remotely operated towed vehicle |
| `sat`	                  | sea acceptance trials |
| `sbp`	                | sub bottom profiler |
| `scs`	                | single channel seismic, sismica monocanal |
| `seism`	              | campanhas de sismica |
| `ssdm`	               | Seabed Survey Data Model |
| `sss`	                | side scan sonar - sonar de varrimento lateral |
| `std-dev`             | desvio padrão |
| `svp`	                | sound velocity profile, dados da velocidade de propagação do som na coluna de água |
| `tf`	                 | dados processados de magnetometro |
| `thick`	              | atributo de espessura de uma unidade sismica |
| `twt`	                | linhas sísmicas com unidades verticais em tempo |
| `uhrs`	               | ultra high resolution seismic, sismica de ulta- alta resolução |
| `usbl`	               | Ultra-short baseline acoustic positioning system |
| `vel-int`	            | velocidades intervalares |
| `vel-picks`	          | pontos de amostragem de velocidade |
| `vel-rms`	            | velocidades root mean square, velocidades de raiz do quadrático médio, extraidas da sismica multicanal |
| `vel-stacking`	       | velocidades usadas para stacking dos dados prestack |
